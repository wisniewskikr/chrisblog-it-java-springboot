DESCRIPTION
-----------

##### Goal
The goal of this project is to present how to create an application in **Java** programming language with usage **Spring Boot** framework which displays **Hello World** message in **JSON** format on a **browser**.

##### Flow
The following flow takes place in this project:
1. User via any browser sends request to application for a content.
1. Application HelloWorld sends back response to user via browser with JSON containing message.

##### Launch
To launch this application please make sure that the **Preconditions** are met and then follow instructions from **Usage** section.

PRECONDITIONS
-------------

##### Preconditions - Tools
* Installed **Operating System** (tested on Windows 10)
* Installed **Java** (tested on version 11.0.16.1)
* Installed **Maven** (tested on version 3.8.5)
* Installed **Git** (tested on version 2.33.0.windows.2)


##### Preconditions - Actions
* **Download** source code using Git with `git clone https://github.com/wisniewskikr/chrisblog-it-java.git`
* Go to **project's folder** and open it with any **Command Line** tool (for instance "Windonw PowerShell" on Windows OS)


USAGE
-----

Usage steps:
1. Start application with `mvn spring-boot:run`
2. Visit `http://localhost:8080`
3. Clean up environment with `ctrl + C`
